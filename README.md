
### 项目说明：
微信小程序：一个可以查看全球最新精品及不同分类商品的商城类小程序


### 目录结构：
- images — 存放项目图片
- pages — 存放项目页面相关文件
- style — 存放独立wxss样式文件，可import引入
- utils — 存放utils文件，可require引入

## 介绍

小程序用到的MemFire Cloud的功能包括：
- 云数据库：存储小程序数据表的信息。
- 即时API：创建数据表时会自动生成 API。

## 主要使用

下载小程序需要的Supabase 小程序客户端。

```
npm install supabase-wechat-stable

```
